• web QQ是本人耗时二个星期完成，属于一个即时通的web桌面应用，还有很多功能需要增加以及优化
  • 使用原生JavaScript进行开发，参照各种主流库自己写了一个微型框架并且可以很好的返回避免命名冲突。
  • 整个网站使用命名空间形式开发，没有全局变量污染，可以很好的协同开发，效果都经过封装，可能有补足的地方但是会慢慢把结构封装的更清晰，降低耦合性并通过类式继承方便复用。
  • 其中采用了css3的一些新功能，在现代浏览器下拥有更好的支持，有时间还会加入推送,定位，离线缓存等新功能
  • 由于时间问题，用户体验细节考虑的不是很到位，交互方面还有一些bug还未处理，以后会一一完善
  • JS操作DOM是降低效率的重大原因，为大量DOM注册事件，渲染浏览器，以及销毁DOM，移除事件等DOM操作还有待优化，可以采用事件委托，记录事件标志，模拟事件等方式处理
  • 后台用的是PHP，应用MVC模式开发实现web即时通讯，实现了GD库画出验证码，无刷新图片上传，预览，在线切割等功能
  • 关于通信这里用的是简单的轮询，长轮询太吃服务器不过现在Comet 确实用处很广，HTML5的EventSource以及采用websocket协议都是利器，但是支持的浏览器太少。

  • （1）点击右侧菜单QQ图标弹出登陆框，enter可以提交，注册后头像为随机分配，如果已经登陆弹出QQ面板
  • （2）提供三组帐号测试，已经互加好友，用--分割帐号密码，8881687--888167;888168--888168;888169--888169;
  • （3）登陆后发送ajax获取好友以及最新消息，弹出QQ面板，可以缩小，ctrl+enter可以快捷发送消息
  • （4）QQ面板左下角可以增加好友，由于需要加好友测试顾暂时没有增加好友策略，加了就直接通过并刷新好友列表获得最新好友
  • （4）获取到消息后头像通过定时器闪动，如果QQ是缩小状态，缩小图标同样闪动，如果点击弹出消息框头像停止闪动
  • （5）消息框通过计算scrooTop让滚动条达到最底显示最新消息，消息框并且控制弹出位置避免重叠,点击显示到最上面
  • （6）由于时间仓促，待完成功能：修改个人资料，处理开发中的细节问题等

  • Desktop APP：

  • （1）APP自动竖排布局：根据可视区的高度，计算出每一个APP的top和left值，然后设置；
  • （2）APP互换位置（缓冲运动效果）：判断距离最近APP，然后让两者的top和left值互相更换；
  • （3）窗口改变大小的时APP重新排列位置：当window.onresize事件触发时，立即重新排列APP
  • （4）并且通过判断鼠标移动APP距离判断是单击打开还是拖拽

  • 设置网站的图标：

    网页标题栏上设置了ico图标

  • 桌面天气挂件:

  • 用Ajax获取各个城市的温度和天气情况，可以拖拽关闭

  • 主题设置:

  • （1）打开对应的弹出窗，点击图片获取获取地址，更换背景图
  • （2）自动用cookie保存背景颜色，刷新后背景不变
  • （3）可以自由上传图片作为背景

  • APP的弹出窗：

  • （1）具有最小化，最大化，还原窗口功能,可双击切换：最小化使用多个运动让效果达到逼真的最小化动画效果；
  • （2）根据点击不同的APP，改变弹出层内容:设置对应的链接内容，达到最好的体验效果，并且可以放大缩小，缩小后底部保留缩小后的栏目
  • （3）继承弹出窗以实现网站多处的不同弹窗,都采用磁性吸附

  • 桌面时钟：

  • （1）用css3制作了桌面的时钟，获取了标准时间，配合css3功能，实现同步加载;
  • （2）此处采用拖拽丢放并且模拟重力，可以随意丢出运动最后停止

  • 右键菜单【刷新】模拟windows刷新功能：

  • 右键菜单【排列方式】：

    用运动的方式上桌面的图片横向，纵向移动，自由排列则可任意摆放其位置，再次设定为横向或纵向时，将重新排列；

  • 右键菜单【主题设置】：

    点击背景图片可以改变桌面的背景

  • 右键菜单【图标设置】：

    可以切换图标，改变桌面图标的大小，以运动的形式移动位置

  • 点击桌面，右键菜单淡出桌面

  • 设置弹出框:

  • （1）自由拖拽（限制在可视区范围）；
  • （2）移入移出的运动效果：让其在打开、关闭时做缓冲运动；
  • （3）弹出框的四周均使用了CSS3中的“border-radius”，这样来设置圆角。
  • （4）新窗口设有最小化、最大化、缩小以及关闭功能，双击标题栏也可缩放，窗口最小化后将在任务栏中出现。

  • 右侧菜单导航

    ：导航是通过计算鼠标到期图片中心距离，得出比例，最近为最大，反之最小，另外设置最小值。

  • 点击焦点切换桌面:

  • （1）点击焦点数字的时候，运动，并改变透明度切换桌面图标
  • （2）用鼠标滚轮同样可以切换桌面

  • 右上角个人信息:

  • （1）运动到桌面根据先后顺序展开【个人信息】，屏幕居中显示
  • （2）点击关闭，收缩并返回先前位置

  • 点击桌面图标：

    显示对应网站的链接

  • 刷新页面或者关闭窗口的窗口提示：

    使用“window.onbeforeunload”即可。

  • 右键菜单的重新设置：

    阻止document.oncontextmenu的默认行为，重新设置右键菜单，让其“top”和“left”跟随鼠标的坐标
